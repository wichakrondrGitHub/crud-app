# crud-app
base on react js api data with [mockapi.io
](https://mockapi.io/)

## Build Setup

```bash
# install project
$ git clone https://github.com/hasankemaldemirci/react-crud-app.git

# open app directory
$ cd react-crud-app

# install dependencies
$ npm i || npm install

# serve with hot reload at localhost
$ npm run dev

# build for production
$ npm build
```
